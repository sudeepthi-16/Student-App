# Student-App
A simple Angular application that displays a list of students and allows adding new students using a reactive form.
**Features**

Home page shows all students (name, class, gender, hobby, favorite subject).
App starts with 2 default students.
Sidebar menu with Home and Add Student options.
Add Student form includes:
Name (required, min 5 chars)
Class dropdown (6â€“9)
Gender (Male/Female)
Has Hobby (checkbox)
Hobby input (shown only if Has Hobby is checked)
Favorite Subject dropdown
Class-based messages shown using *ngIf.
On Save, student is added to the list and user returns to Home page.
